## Generated Artifacts

Generated topology docs live under:

- `core/scripts/cre/environment/docs/TOPOLOGIES.md`
- `core/scripts/cre/environment/docs/topologies/`

They are generated by:

```bash
cd core/scripts/cre/environment
go run . topology generate
```

## Main Environment Commands

```bash
go run . env setup
go run . env start
go run . env stop
go run . env restart
go run . env workflow deploy
go run . env workflow delete
go run . topology list
go run . topology show
go run . topology generate
```
